Splits a ".mov" regarding an ".edl" file
-----------------------------------------

Prerequisites:
----------------------
-win 64
-ffmpeg (provided in "ffmpeg" folder)
-a ".mov" file (initially encoded in h264, but it should work whatever the encoding)
-an ".edl" file, with the exact same name (except the extension...of course), and located in the same folder
-python 2.7+ (< 3.0), reachable at : "C:\Python27\python.exe" (core-coded in "splitAll.bat")

Usage:
----------------------
Simple way : Drag-Drop a ".mov" on "splitAll.bat"
